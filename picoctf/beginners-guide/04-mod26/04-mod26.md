## Objective

Decode a ROT13-encoded string to find the flag.

---

## Solution

Downloaded the file:

    wget .../values.txt
    cat values.txt

Contents:

    cvpbPGS{arkg_gvzr_V'yy_gel_2_ebhaqf_bs_ebg13_45559noq}

Decoded using tr:

    cat values.txt | tr 'A-Za-z' 'N-ZA-Mn-za-m'

Output:

    picoCTF{next_time_I'll_try_2_rounds_of_rot13_45559abd}

---

## Concepts Learned

ROT13 is a substitution cipher that shifts every letter 13 positions in the alphabet. Since the alphabet has 26 letters, applying ROT13 twice returns the original text. It is not encryption -- it provides no security. It is commonly used in CTFs as a basic encoding challenge.

The tr command performs character-by-character translation. The pattern 'A-Za-z' mapped to 'N-ZA-Mn-za-m' replaces every letter with its ROT13 equivalent.

CyberChef can also decode ROT13 using the ROT13 operation.

---

## Key Takeaway

ROT13 and other simple substitution ciphers appear constantly in CTFs. Recognising them on sight and knowing how to decode them quickly -- via tr, CyberChef, or Python -- saves time in competitions.
