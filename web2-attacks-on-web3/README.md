# How Web2 Infrastructure Attacks Target Web3 Systems

**Author:** Vektasafe  
**Category:** Cybersecurity Research  

---

## Summary

Approximately 80% of funds stolen from Web3 projects originate from attacks on surrounding Web2 infrastructure -- not smart contract exploits. This document covers six primary attack classes with real incidents and concrete mitigations for each.

---

## Attack Classes Covered

| # | Attack Class | Real Incident | Loss |
|---|-------------|---------------|------|
| 1 | DNS Hijacking | Curve Finance (2022) | 570,000 USD |
| 2 | BGP Hijacking | MyEtherWallet (2018) | 17,000,000 USD |
| 3 | Frontend Repository Compromise | Ledger Connect Kit (2023) | 600,000 USD |
| 4 | API Key and Credential Exposure | Ronin Network (2022) | 625,000,000 USD |
| 5 | Phishing and Social Engineering | Axie Infinity (2022) | 625,000,000 USD |
| 6 | Cloud Infrastructure Misconfiguration | Mixin Network (2023) | 200,000,000 USD |
